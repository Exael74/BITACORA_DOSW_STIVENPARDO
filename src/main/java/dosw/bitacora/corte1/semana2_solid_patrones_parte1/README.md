# Semana 2: SOLID y Patrones de Software Parte 1

## Objetivo
Introducción a los principios SOLID y primeros patrones de diseño de software. Implementación de patrones creacionales, estructurales y de comportamiento.

## Contenido

### Ejercicios de tarea en casa

#### Ejercicio 1 - Patrón Creacional (Factory Method)
- **ejercicio1_factory_method/**: Implementación del patrón Factory Method
- Creación de objetos sin especificar sus clases concretas

#### Ejercicio 2 - Patrón Estructural (Adapter)  
- **ejercicio2_adapter/**: Implementación del patrón Adapter
- Compatibilidad entre interfaces incompatibles

#### Ejercicio 3 - Patrón de Comportamiento (Memento)
- **ejercicio3_memento/**: Implementación del patrón Memento
- Captura y restauración del estado de objetos

#### Ejercicio 4 - Principios SOLID (Calculadora simple)
- **ejercicio4_solid_calculadora/**: Aplicación de principios SOLID
- Calculadora que respeta los 5 principios SOLID

## Skills desarrolladas
