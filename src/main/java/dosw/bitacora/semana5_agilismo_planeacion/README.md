# Semana 5: Agilismo y Planeación de Trabajo

## Objetivo
Implementación de metodologías ágiles para la gestión de proyectos de software, uso de herramientas de planificación y seguimiento de tareas.

## Contenido

### Ejercicio realizado en clase

#### Elaboración del Agiaco con JIRA
- **agiaco_jira/**: Proyecto integral utilizando metodología ágil
- Implementación de SCRUM/Kanban
- Gestión de backlog y sprints
- Seguimiento de métricas ágiles

## Skills desarrolladas
- Metodologías ágiles (SCRUM, Kanban)
- Planificación de sprints
- Gestión de backlog de producto
- Estimación ágil (Planning Poker, Story Points)
- Seguimiento y control de proyectos
- Uso de herramientas de gestión (JIRA)

## Conceptos aplicados
- **SCRUM Framework**:
  - Product Owner
  - Scrum Master  
  - Development Team
  - Sprint Planning
  - Daily Standup
  - Sprint Review
  - Sprint Retrospective

- **Artefactos SCRUM**:
  - Product Backlog
  - Sprint Backlog
  - Increment

## Herramientas utilizadas
- JIRA Software
- Tableros Kanban/SCRUM
- Burndown Charts
- Velocity Charts

## Metodología
- Implementación práctica de SCRUM
- Configuración de proyecto en JIRA
- Gestión de historias de usuario
- Seguimiento de métricas de equipo
- Análisis de velocidad y mejora continua
