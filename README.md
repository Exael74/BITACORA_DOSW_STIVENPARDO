# Bitácora Técnica DOSW - Desarrollo de Software

## Descripción del Proyecto
Bitácora técnica para el seguimiento y documentación de los ejercicios, proyectos y aprendizajes del curso de Desarrollo de Software (DOSW).

## Estructura del Repositorio

### 📁 Semana 1 - Streams y Git Flow
**Ubicación**: [src/main/java/dosw/bitacora/semana1_streams_gitflow/](src/main/java/dosw/bitacora/semana1_streams_gitflow/)

**Contenido**:
- **Ejercicios de repaso de streams**: 3 ejercicios realizados en clase
- **Ejercicios de tarea**: 2 ejercicios para casa
- **Herramientas**: Java Streams API, Git Flow

### 📁 Semana 2: SOLID y Patrones de Software Parte 1
**Ubicación**: [src/main/java/dosw/bitacora/semana2_solid_patrones_parte1/](src/main/java/dosw/bitacora/semana2_solid_patrones_parte1/)

**Ejercicios de tarea en casa**:
- ✅ **Ejercicio 1**: Patrón Creacional (Factory Method)
- ✅ **Ejercicio 2**: Patrón Estructural (Adapter)
- ✅ **Ejercicio 3**: Patrón de Comportamiento (Memento)
- ✅ **Ejercicio 4**: Principios SOLID (Calculadora simple)

### 📁 Semana 3: SOLID y Patrones de Software Parte 2
**Ubicación**: [src/main/java/dosw/bitacora/semana3_solid_patrones_parte2/](src/main/java/dosw/bitacora/semana3_solid_patrones_parte2/)

**Ejercicios realizados en clase**:
- 🏭 **Factory Method**: Comercio electrónico
- 🏭 **Abstract Factory**: Empresa de Videojuegos (Xbox y PlayStation)
- 🔧 **Builder**: Fábrica de juguetes
- 🔌 **Adapter**: Gasolinería Inteligente
- 🌉 **Bridge**: Figuras y Colores
- 🌳 **Composite**: Bodega de productos
- 🎨 **Decorator**: Simulador de batalla naval
- ⛓️ **Chain of Responsibility**: Embajada de estados unidos
- 💻 **Command**: Personajes de videojuegos
- 🔄 **Iterator**: Viaje a roma (diagrama)
- 📊 **Strategy**: Aplicación de navegación

### 📁 Semana 4: Análisis de Requerimientos
**Ubicación**: [src/main/java/dosw/bitacora/semana4_analisis_requerimientos/](src/main/java/dosw/bitacora/semana4_analisis_requerimientos/)

**Ejercicios realizados en clase**:
- 🚗 **Diagrama de contexto de UBER**
- 📝 **Ejercicio análisis de requerimientos**: Formulario de inscripción a la carrera
  - Requerimiento detallado completo (Nombre, Descripción, Actor Principal, etc.)
  - Diagrama de casos de uso
  - Reglas de negocio
- 🆔 **Manual de Identidad del Estudiante**

### 📁 Semana 5: Agilismo y Planeación de Trabajo
**Ubicación**: [src/main/java/dosw/bitacora/semana5_agilismo_planeacion/](src/main/java/dosw/bitacora/semana5_agilismo_planeacion/)

**Ejercicio realizado en clase**:
- 📋 **Elaboración del Agiaco con JIRA**
- Implementación de metodologías ágiles
- Gestión de proyectos con herramientas especializadas

## Tecnologías Utilizadas
- ☕ **Java**: Lenguaje principal de desarrollo
- 🏗️ **Maven**: Gestión de dependencias y construcción
- 🔄 **Git Flow**: Control de versiones
- 📊 **JIRA**: Gestión ágil de proyectos
- 🎯 **UML**: Modelado de software

## Competencias Desarrolladas

### Técnicas
- Programación orientada a objetos avanzada
- Patrones de diseño de software
- Principios SOLID
- Análisis de requerimientos
- Metodologías ágiles (SCRUM/Kanban)

### Herramientas
- Java Streams API
- Git Flow para versionado
- JIRA para gestión de proyectos
- Diagramación UML
- Documentación técnica

## Cómo Navegar la Bitácora
1. Cada semana tiene su carpeta específica con descripción clara
2. Cada ejercicio está en su propia subcarpeta con nombre descriptivo
3. Consulte el README.md de cada semana para información detallada
4. Los archivos Java están organizados por tema y tipo de ejercicio
