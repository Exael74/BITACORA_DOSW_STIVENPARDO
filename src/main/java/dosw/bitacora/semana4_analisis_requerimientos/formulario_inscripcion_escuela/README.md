# Análisis de Requerimientos - Formulario de Inscripción a la Carrera

## Descripción
Análisis completo de requerimientos para el desarrollo de un sistema de formulario de inscripción a carreras universitarias.

## Documentación Requerida

### 1. Requerimiento Detallado
- **Nombre**: Gestión de Inscripción a Carreras
- **Descripción**: Sistema que permite a estudiantes inscribirse a carreras académicas
- **Cómo se ejecutará**: A través de plataforma web con formulario estructurado
- **Actor Principal**: Estudiante aspirante
- **Precondiciones**: 
  - Estudiante debe tener bachillerato completo
  - Sistema debe estar disponible en período de inscripciones
- **Datos de entrada**: Información personal, académica y de contacto
- **Datos de salida**: Número de solicitud, confirmación de recepción

### 2. Flujos del Sistema
- **Flujo Básico**: Proceso normal de inscripción exitosa
- **Flujo Alterno (Error)**: Manejo de errores de validación, sistema no disponible

### 3. Artefactos
- **Diagrama de Casos de Uso**: Representación visual de interacciones
- **Reglas de negocio**: Políticas institucionales y restricciones
