# Elaboración del Agiaco con JIRA

## Descripción
Proyecto integral para la implementación de metodologías ágiles utilizando JIRA como herramienta de gestión.

## Objetivos
- Configurar un proyecto ágil en JIRA
- Implementar SCRUM framework completo
- Gestionar backlog de producto y sprints
- Realizar seguimiento de métricas ágiles

## Componentes del Proyecto

### 1. Configuración Inicial
- Creación del proyecto en JIRA
- Configuración de workflows
- Definición de tipos de issues
- Configuración de tablero SCRUM

### 2. Gestión de Backlog
- Creación de épicas
- Definición de historias de usuario
- Estimación con Story Points
- Priorización de backlog

### 3. Sprint Management
- Planificación de sprints
- Ejecución y seguimiento diario
- Sprint Review y Retrospective
- Métricas de velocidad

### 4. Roles SCRUM
- **Product Owner**: Definición de requerimientos
- **Scrum Master**: Facilitación del proceso
- **Development Team**: Implementación

## Artefactos a Desarrollar
- `Configuracion_Proyecto_JIRA.md` - Setup inicial
- `Backlog_Producto.md` - Lista priorizada de features
- `Historias_Usuario.md` - User stories detalladas
- `Sprint_Planning.md` - Documentación de planificación
- `Metricas_Agiles.md` - Burndown charts y velocity
- `Retrospectivas.md` - Lecciones aprendidas

## Herramientas Utilizadas
- JIRA Software
- Confluence (documentación)
- Tableros Kanban/SCRUM
- Reporting y métricas ágiles
